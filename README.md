Cross-platform Flutter app: pick subtitles from OpenSubtitles and read them
aloud via TTS in sync with streaming video.

**For those who cannot read, and for those learning a new language.**

## Quick Start

```bash
flutter pub get
flutter run
```

## Download

Grab the latest build from [Releases](https://github.com/nmwael/subvocal/releases).

## What it does

- Picks up subtitles and reads them aloud in real time, synchronized with your video
- Translates subtitles on the fly into your language (including Danish, Spanish, Galician)
- Helps language learners hear pronunciation while reading along
